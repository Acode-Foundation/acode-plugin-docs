# Terminal Settings

Configure the built-in terminal emulator.

## Permissions

| Setting              | Description                                                                                                                                |
| :------------------- | :----------------------------------------------------------------------------------------------------------------------------------------- |
| **All Files Access** | Grants full internal storage access to be accessible from terminal. Only effective on Android 10+ and not available on Play Store version. |

## Appearance

| Setting                   | Description                                     |
| :------------------------ | :---------------------------------------------- |
| **Font Size**             | Adjust the terminal font size.                  |
| **Font Family**           | Choose a font for the terminal.                 |
| **Theme**                 | Select a color scheme for the terminal.         |
| **Cursor Style**          | Choose cursor shape: block, underline, or bar.  |
| **Cursor Inactive Style** | Cursor appearance when terminal is not focused. |
| **Font Weight**           | The font weight used to render non-bold text.   |
| **Cursor Blink**          | Enable or disable cursor blinking animation.    |
| **Font Ligatures**        | Whether font ligatures are enabled in terminal. |
| **Letter Spacing**        | The spacing in whole pixels between characters. |

## Behavior

| Setting                        | Description                                                                                                                              |
| :----------------------------- | :--------------------------------------------------------------------------------------------------------------------------------------- |
| **Scrollback Lines**           | The amount of scrollback in the terminal. Scrollback is the amount of rows retained when lines are scrolled beyond the initial viewport. |
| **Tab Stop Width**             | The size of tab stops in the terminal.                                                                                                   |
| **Convert EOL**                | Convert end-of-line characters.                                                                                                          |
| **Image Support**              | Whether images are supported in the terminal.                                                                                            |
| **Confirm Terminal Tab Close** | Ask for confirmation before closing terminal tabs.                                                                                       |

## Installation Management

| Action        | Description                                     |
| :------------ | :---------------------------------------------- |
| **Backup**    | Creates a backup of the terminal installation.  |
| **Restore**   | Restores a backup of the terminal installation. |
| **Uninstall** | Uninstalls the terminal installation.           |
