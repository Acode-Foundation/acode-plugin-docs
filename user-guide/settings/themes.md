# Themes

Customize the visual appearance of Acode.

## Overview

The Themes page has two tabs:

| Tab        | Description                                                                             |
| :--------- | :-------------------------------------------------------------------------------------- |
| **App**    | Changes the styling and colors of app components, widgets, sidebar, tabs, dialogs, etc. |
| **Editor** | Changes syntax highlighting colors for code in the editor.                              |

## App Themes

App themes control the overall look of the Acode interface. Available options include:

- Several built-in themes (light and dark variants)
- **System**: Automatically matches your device's theme setting
- Additional themes from plugins

## Editor Themes

Editor themes control syntax highlighting for code. You can choose from many built-in themes or install more via plugins.

::: tip
Many additional themes (both App and Editor) can be found in the [Plugin Registry](/user-guide/plugins).
:::
